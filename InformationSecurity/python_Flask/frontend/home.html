欢迎使用隐私集合操作系统

这是一个基于 Flask 后端的 Web 应用，提供以下功能：

  • 用户注册与登录（JWT 认证）
  • 隐私求交 (PSI) - 双方安全计算共同元素
  • 隐私求并 (PSU) - 输出完整并集, 双方对称可见明文
  • 隐私集合匹配 (PSI-Card) - 子集判定, 返回交集基数与缺失数
  • 隐私求和 (PSI-Sum) - 交集元素的关联数值求和
  • 多方隐私求交 (SS-PSI) - 四方场景求共同交集
  • 管理员用户管理功能

请点击左侧导航栏使用各项功能。
